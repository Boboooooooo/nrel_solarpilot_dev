Page Overview


SolarPILOT can model multiple heliostat geometries in a single layout. You can manage the heliostat templates and their positioning in the field on the template management page. This page appears in the page list as Heliostats.


Two input groups are available.


Heliostat Templates


Heliostat Type Use


Heliostat Templates

- Top of page -


The Heliostat Templates group allows you to add, delete, disable, or rename heliostat templates. Actions taken in this group affect the templates that appear in the page list.



Add New [Button]

Add a new heliostat template to the system. This action will create a new heliostat geometry template that will appear in the page list. The default values for the new template are copied from the first template in the template list. Once created, the geometry and properties of the heliostat template can be modified by navigating to the newly created page and specifying values as needed.


When adding a new template, you will be prompted to provide a name for the template. You must use a unique name for each template.


Delete [Button]

Delete the selected heliostat template from the template list. This button has no action unless a template is selected. Once confirmed, the action cannot be undone.


Disable [Button]

Disable a selected heliostat template. This button has no action unless a template is selected. A disabled template remains a part of the project file, but is not used as part of the heliostat field layout.


Disabling a heliostat template may be desired when considering multiple heliostat options as part of a field layout, for example.


Rename [Button]

Rename the selected template.


Move Item [Buttons]

The order of the heliostat templates in the lists can be important, depending on the Heliostat Geometry Distribution option that you select. For more information on how order may affect your system, refer to Heliostat Geometry Distribution notes.


Template [List Control]

The heliostat template list provides a full list of all heliostat templates that are currently part of the project file. The template name and template state are provided. The template state can be modified using the action buttons described above.



Heliostat Type Use

- Top of page -


The Heliostat Type Use group determines how multiple heliostat templates (if applicable) are distributed in the heliostat field layout. Because SolarPILOT can model multiple heliostat geometries, rules for the positioning of each heliostat type are governed in this input group.



Heliostat Geometry Distribution

Select the heliostat geometry (template) spatial distribution method.


Use Single Template

With this option, all heliostats in the field are based on the geometry defined in a single template.


Select Heliostat Geometry

Select the heliostat geometry template to be used for the heliostat field layout. The drop-down list contains all active heliostat geometry templates in the template list.


Specified Range

Specify the radial and azimuthal ranges in which each template may be placed. Specify the range requirements for each active template in the provided input table.


Note that:

1) Heliostats not contained within any specified range will assume the first active template in the list.

2) Heliostats contained in multiple ranges will use the first available template in the list.


The range extents are defined as:

-180 = +180 = South

angle < 0 = West

angle >= 0 = East

0 = North


Even Radial Distribution

The active heliostat templates are assigned to evenly-spaced radial zones in the order specified in the template list control. The radial range for each template is equal to the maximum allowable positioning radius minus the minimum allowable positioning radius all divided by the number of heliostat templates.
